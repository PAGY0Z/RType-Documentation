AssetsManagerExceptionsType Enumeration.

The AssetsManagerExceptionsType enumeration defines the types of exceptions that can be thrown by the AssetsManager class template.

Note
The enumeration values are as follows:
- ERROR_AT_LOADING_TEXTURE: Error at loading a texture
- ERROR_AT_LOADING_FONT: Error at loading a font
- ERROR_AT_LOADING_MUSIC: Error at loading a music
- ERROR_AT_LOADING_SOUND: Error at loading a sound

AssetsManagerExceptionsMessage Structure.

The AssetsManagerExceptionsMessage structure defines the message associated with each type of exception that can be thrown by the AssetsManager class template.

Note
The structure includes the following members:
- __type: The type of exception
- __message: The message associated with the exception
