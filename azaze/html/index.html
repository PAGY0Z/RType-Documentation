🌐 Description

The asset structure is designed to help you easily find and use the resources required for the R-Type project. Each folder is dedicated to a specific type of asset, ensuring a clean and efficient workflow.

💼 Usage

If you want to add custom assets, we recommend placing each file in the appropriate directory. This improves the organization and readability of your project.

🔧 <strong>Bonus Tool</strong>

For configuration files, you can always use our provided SpriteSheetGenerator.py. It’s a great tool for handling complex or generic sprite sheets effortlessly.
