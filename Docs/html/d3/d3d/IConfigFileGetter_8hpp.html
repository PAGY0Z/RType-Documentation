This file contains the definition of the IConfigFileGetter interface. The IConfigFileGetter interface provides methods for retrieving configuration information from various file formats.

The IConfigFileGetter interface defines methods for retrieving information about spritesheets, actions, and specials from configuration files. It also provides methods for checking file extensions, verifying the existence of paths, and ensuring directories are not empty.

Example:
#include "IConfigFileGetter.hpp"
class ConfigFileGetter : public IConfigFileGetter {
public:
bool GetInfos(const std::string &path, const std::shared_ptr<Spritesheet> &spritesheet) override;
bool GetActions(const std::string &path, const std::shared_ptr<Spritesheet> &spritesheet) override;
bool GetSpecials(const std::string &path, const std::shared_ptr<Spritesheet> &spritesheet) override;
bool CheckExtension(const std::string &path) override;
bool CheckSpritesheetAssets(const std::string &path) override;
bool CheckDirectoryNotEmpty(const std::string &path) override;
bool CheckPathExist(const std::string &path) override;
bool IsDirectory(const std::string &path) override;
};
