This file contains the definition of the Spritesheet class. The Spritesheet class is responsible for managing and manipulating spritesheets, including setting dimensions, adding frames, actions, and durations.

The Spritesheet class provides methods to set the dimensions of the spritesheet, add frames, actions, and durations, and update frames. It also provides methods to get the dimensions of the spritesheet, get specific frames and actions, and check if the spritesheet is valid.

Example:
#include "Spritesheet.hpp"
int main()
{
Spritesheet spritesheet("example_spritesheet");
spritesheet.SetWidth(1024);
spritesheet.SetHeight(768);
Frame frame = {0, 0, 64, 64};
spritesheet.AddFrame(1, frame);
std::vector<int> action_frames = {1, 2, 3};
spritesheet.AddAction("walk", action_frames);
spritesheet.Display();
return 0;
}
