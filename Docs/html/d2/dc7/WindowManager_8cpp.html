This file contains the implementation of the WindowManager class. The WindowManager class is responsible for managing the main application window, including creating, closing, and providing access to the window.

The WindowManager class uses the SFML library to create and manage the main application window. It provides methods to create the window, close the window, and get a reference to the window.

Example:
#include "WindowManager.hpp"
int main()
{
WindowManager windowManager;
while (windowManager.getWindow().isOpen()) {
// Handle events and rendering
}
windowManager.closeWindow();
return 0;
}
