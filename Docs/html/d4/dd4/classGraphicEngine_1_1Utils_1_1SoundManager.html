Detailed Description

SoundManager Class.

The SoundManager class is a utility class that provides a way to manage sounds and musics. The SoundManager class provides methods to play, stop, pause, and resume sounds and musics. The class is designed to be easy to use and extend, making it an ideal module for new graphic SFML projects.
