Detailed Description

The GraphicEngine class.

The GraphicEngine class is the main class of the GraphicEngine library. It is used to create a window and draw shapes, sprites, and text on the screen. The GraphicEngine class provides a simple interface to the SFML library, allowing you to create a window, draw shapes, sprites, and text on the screen, and handle events. The GraphicEngine class is designed to be easy to use and extend.

The GraphicEngine class is the main class of the GraphicEngine library. It is used to create a window and draw shapes, sprites, and text on the screen. The GraphicEngine class provides a simple interface to the SFML library, allowing you to create a window, draw shapes, sprites, and text on the screen, and handle events. The GraphicEngine class is designed to be easy to use and extend. It offuscates the complexity of the SFML library and provides a simple interface to draw shapes, sprites, and text on the screen. The GraphicEngine class is organized into a number of directories, each containing a different set of classes and functions. The GraphicEngine class uses a private implementation to hide and offuscate the SFML library.

Example:
#include "GraphicEngine.hpp"
auto graphicEngine = GraphicEngine::GraphicEngine(1920, 1080, "My Graphic Project");
auto circle = std::make_shared<GraphicEngine::CircleShapeDrawable>();
circle->SetPosition({100, 100});
circle->SetRadius(50);
circle->SetFillColor({255, 0, 0, 255});
std::vector<GraphicEngine::Event> events;
while (graphicEngine.IsOpen()) {
events = graphicEngine.GetEvents();
for (auto &event : events) {
if (event.type == GraphicEngine::Event::Closed) {
graphicEngine.CloseWindow();
}
}
graphicEngine.AddDrawable(circle);
graphicEngine.RenderWindow();
}
