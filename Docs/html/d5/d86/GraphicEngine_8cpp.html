This file contains the implementation of the GraphicEngine Library. The GraphicEngine Library is a template library that can be used as a starting point for new graphic projects. The GraphicEngine Library contains a number of utility classes and functions that can be used to simplify the development of new graphic projects. The GraphicEngine Library is organized into a number of directories, each containing a different set of classes and functions. The GraphicEngine Library is designed to be easy to use and extend, making it an ideal starting point for new graphic projects.

The GraphicEngine Library is designed to be easy to use and extend, making it an ideal starting point for new graphic projects. It offuscates the complexity of the SFML library and provides a simple interface to draw shapes, sprites, and text on the screen.

Example:

auto graphicEngine = GraphicEngine::GraphicEngine(1920, 1080, "My Graphic Project");
