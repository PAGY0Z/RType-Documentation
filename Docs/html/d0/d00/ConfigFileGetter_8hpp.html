This file contains the declaration of the ConfigFileGetter class. It provides methods to load configuration file paths, retrieve spritesheets, and display their information.

The ConfigFileGetter class is responsible for loading configuration file paths from a specified directory, retrieving the spritesheets, and displaying their information using the Display method of the Spritesheet class.

Example:
int main() {
std::string spriteSheetDir = "../Assets/spritesheets";
std::string rootPath = "../Assets";
ConfigFileGetter configGetter(spriteSheetDir, rootPath);
auto spritesheets = configGetter.GetSpritesheets();
for (const auto &spritessheets : spritesheets) {
spritessheets->Display();
}
return 0;
}
